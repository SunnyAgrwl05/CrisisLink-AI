CRISISLINK AI — DEMO SCRIPT (3-4 min)
=======================================

[0:00–0:20] HOOK
"In the first hours after a disaster, fake SOS calls waste rescue teams,
agencies work blind, and citizens can't find shelter. CrisisLink AI fixes
that with a multi-agent Google ADK pipeline."

[0:20–0:45] ARCHITECTURE (show assets/architecture_diagram.svg)
"One citizen report goes through 8+ specialized agents — SOS verification,
damage assessment, priority scoring, resource allocation, shelter and
hospital lookup — all coordinated by a Google ADK Workflow graph, backed
by an MCP server for live weather, maps, and inventory data."

[0:45–1:00] RUN IT
Terminal:
    make playground

[1:00–1:30] INPUT THE DEMO SCENARIO
Paste exactly:
    "There is severe flooding in Patna. 15 people trapped near Gandhi Setu."

Narrate while it runs:
"Orchestrator extracts the location and hazard type. SOS Verification and
Damage Assessment run in parallel — SOS confirms this isn't spam, Damage
checks live weather and rates this Critical."

[1:30–2:00] PRIORITY + PARALLEL AGENTS
"Priority Agent scores this ~90/100 — Critical. That triggers three agents
in parallel: Resource Allocation reserves ambulances and a boat from
live inventory, Shelter Agent finds Rajendra Nagar Relief Camp, Medical
Agent finds PMCH with ICU beds and gives safe first-aid guidance."

[2:00–2:25] SECURITY CHECKPOINT
"Before anything goes out, the Security Checkpoint — plain code, not an
LLM — masks any PII and scans for prompt injection. Show audit_log.jsonl
here: every decision is logged with a timestamp and level."

[2:25–2:50] HUMAN APPROVAL
"Because this is Critical, a human dispatcher must approve before
citizens and authorities are notified — CrisisLink assists, it doesn't
replace judgment on life-and-death calls."
Show the approval prompt appearing in the terminal/adk web UI.

[2:50–3:15] FINAL OUTPUT
Show the final JSON: citizen_message, authority_summary, ngo_report,
sms_summary — read the citizen_message aloud.

[3:15–3:30] CLOSE
"CrisisLink AI: verified, prioritized, resourced, and secured rescue
coordination — in the minutes that matter most. Built on Google ADK,
MCP, and Gemini."
